🤚 Gesture-Controlled Robotics System

> Control a robot using hand gestures via MediaPipe and computer vision.

## Tech Stack
- Python
- MediaPipe
- OpenCV
- Arduino / ESP32
- PyFirmata

## Features
- Real-time hand gesture recognition
- Robot movement via gestures
- Low-latency control system

## Project Structure
- src/    — Source code
- docs/   — Documentation
- media/  — Photos and videos
