# Wiring Guide - Gesture Controlled Robot

## USB Camera -> Raspberry Pi 4
| Camera | RPi      |
|--------|----------|
| USB    | USB Port |

## Motor Driver L298N -> Raspberry Pi 4
| L298N | RPi GPIO |
|-------|----------|
| IN1   | GPIO 17  |
| IN2   | GPIO 18  |
| IN3   | GPIO 22  |
| IN4   | GPIO 23  |
| GND   | GND      |
| VCC   | 5V       |
